# World Faults and Earthquakes

**Live:** https://arashnassirpour.com/world-faults/

An interactive globe showing the world's major tectonic plate boundaries
(colour-coded by mechanism), 22 named fault systems, and significant
earthquakes from the past 30 days pulled live from USGS.

Built as a single self-contained `index.html`. Hosted on GitHub Pages; any commit
to `main` goes live automatically. Plate boundary data: Bird (2003) PB2002
model. Made for learning and exploration, not for hazard assessment.
